# 腾讯云转码工具 - 分发说明

## 如何分发给其他用户

### 推荐方式：打包发送以下文件

1. **腾讯云转码工具.app** - 主应用程序
2. **使用说明.md** - 使用说明文档（重要！）
3. **快速修复.command** - 快速修复脚本（可选，但推荐）

### 分发步骤

1. 进入 `dist/` 目录
2. 复制以下文件：
   - `腾讯云转码工具.app`
   - `使用说明.md`（从项目根目录复制）
   - `快速修复.command`（从项目根目录复制，可选）

3. 将这些文件打包成 ZIP 压缩包发送，或者直接发送文件夹

### 用户收到后的操作

用户应该：
1. 解压文件（如果是 ZIP 包）
2. **首先阅读 `使用说明.md`**
3. 如果应用无法打开，双击 `快速修复.command` 文件
4. 或者按照使用说明中的其他方法操作

## 注意事项

⚠️ **重要提示**：
- macOS 会标记从网络下载的应用
- 即使用户本地打包时移除了隔离属性，通过网络传输后可能会重新添加
- 因此**强烈建议**同时发送 `使用说明.md` 和 `快速修复.command`

## 替代方案：创建 DMG 安装包

如果需要更专业的分发方式，可以创建 DMG 安装包：

```bash
# 需要安装 create-dmg（可选）
# brew install create-dmg

# 创建 DMG
create-dmg \
  --volname "腾讯云转码工具" \
  --window-pos 200 120 \
  --window-size 800 400 \
  --icon-size 100 \
  --icon "腾讯云转码工具.app" 200 190 \
  --hide-extension "腾讯云转码工具.app" \
  --app-drop-link 600 185 \
  "腾讯云转码工具.dmg" \
  "dist/"
```
