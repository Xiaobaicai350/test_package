# 第三阶段：实战项目（静态网页）

## 🎯 学习目标

1. 综合运用前两阶段知识
2. 完成实际的爬虫项目
3. 学习数据存储（CSV、JSON、数据库）
4. 掌握爬虫项目的完整流程

## 📖 项目列表

### 项目1：新闻爬虫（01_news_crawler.py）
**技术栈**：requests + BeautifulSoup
- 爬取新闻列表
- 进入详情页获取完整内容
- 数据清洗
- 保存为JSON

**学习重点**：
- 列表页+详情页爬取模式
- 翻页处理
- 数据结构设计

### 项目2：电商爬虫（02_ecommerce_crawler.py）
**技术栈**：requests + XPath
- 商品列表爬取
- 价格、评分、销量提取
- 图片下载
- 数据分析

**学习重点**：
- 数据提取技巧
- 异常处理
- 进度显示

### 项目3：数据存储（03_data_storage.py）
**技术栈**：pandas + pymysql
- CSV存储
- Excel存储
- JSON存储
- MySQL存储

**学习重点**：
- 各种存储方式对比
- 数据库操作
- 数据导入导出

## 🔥 项目特色

- ✅ 真实可运行的代码
- ✅ 详细的注释说明
- ✅ 完整的错误处理
- ✅ 可扩展的架构设计

## 💡 项目流程

```
1. 分析目标网站
   ├── 查看网站结构
   ├── 确定数据位置
   └── 制定爬取策略

2. 编写爬虫代码
   ├── 发送请求
   ├── 解析数据
   ├── 数据清洗
   └── 异常处理

3. 数据存储
   ├── 选择存储方式
   ├── 设计数据结构
   └── 保存数据

4. 测试优化
   ├── 测试爬虫稳定性
   ├── 优化性能
   └── 添加日志
```

## ⚠️ 注意事项

1. **遵守robots.txt**：检查网站的爬虫协议
2. **控制频率**：添加延迟，避免给服务器造成压力
3. **User-Agent**：伪装成浏览器
4. **法律合规**：不爬取敏感信息，遵守法律法规
5. **仅供学习**：本教程仅用于学习，请勿用于商业用途
