# 第二阶段：网页解析技术

## 🎯 学习目标

1. 理解HTML/CSS/XPath基础
2. 掌握BeautifulSoup解析库
3. 学会使用XPath和CSS选择器
4. 掌握正则表达式提取数据

## 📖 网页解析技术对比

| 技术 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| BeautifulSoup | 简单易用，容错性好 | 速度较慢 | 一般网页解析 |
| lxml + XPath | 速度快，功能强大 | 学习曲线陡 | 大量数据解析 |
| 正则表达式 | 灵活，无需解析DOM | 复杂难维护 | 简单文本提取 |

## 📝 学习内容

### 1. HTML基础（快速入门）
- HTML标签结构
- 常见标签：div, span, a, img, table
- 属性：class, id, href, src

### 2. BeautifulSoup解析（01_beautifulsoup.py）
- 安装和基本使用
- 查找元素：find(), find_all()
- CSS选择器：select()
- 遍历DOM树

### 3. XPath解析（02_xpath.py）
- XPath语法基础
- 路径表达式
- 条件筛选
- 实战案例

### 4. 正则表达式（03_regex.py）
- 正则基础语法
- 常用模式
- 提取数据
- 实战应用

## 🔥 实战练习

完成以下任务：
1. 解析新闻网站，提取标题、作者、时间、内容
2. 解析电商页面，提取商品名称、价格、评分
3. 解析表格数据，转换为结构化格式

## 💡 技巧总结

1. **优先使用CSS选择器**：简洁易读
2. **XPath适合复杂场景**：处理层级关系
3. **正则表达式作为补充**：提取特定格式数据
4. **调试技巧**：使用浏览器开发者工具（F12）
