# 第一阶段：Python基础与HTTP协议

## 🎯 学习目标

1. 掌握Python基础语法（针对Java开发者）
2. 理解HTTP协议的工作原理
3. 学会使用requests库发送HTTP请求
4. 处理响应数据和异常

## 📖 知识点对比：Java vs Python

| 特性 | Java | Python |
|------|------|--------|
| 类型声明 | 强制声明类型 | 动态类型 |
| 缩进 | 使用大括号{} | 使用缩进 |
| 字符串 | String类 | str类型，更灵活 |
| 集合 | List, Map, Set | list, dict, set |
| 异常处理 | try-catch | try-except |

## 💡 常见问题

**Q: Python和Java在爬虫方面有什么区别？**
A: Python拥有更丰富的爬虫库，代码更简洁，特别适合快速开发和原型验证。

**Q: requests库和urllib的区别？**
A: requests是对urllib的高级封装，API更友好，使用更简单。
