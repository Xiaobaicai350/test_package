# 第四阶段：动态网页与反爬虫

## 🎯 学习目标

1. 掌握动态网页爬取技术
2. 学习Selenium自动化
3. 处理Ajax异步请求
4. 应对常见反爬虫策略

## 📖 核心概念

### 静态网页 vs 动态网页

**静态网页**：
- HTML内容直接在源代码中
- 使用requests即可获取
- 查看源代码能看到数据

**动态网页**：
- 内容由JavaScript动态生成
- 需要浏览器渲染
- 源代码中看不到数据
- 数据通过Ajax异步加载

### 解决方案

1. **Selenium**：模拟真实浏览器
2. **Playwright**：新一代自动化工具
3. **分析Ajax接口**：直接请求API

## 📝 学习内容

- 安装和配置
- 浏览器驱动
- 基本操作（点击、输入、滚动）

### 2. Ajax数据爬取（02_ajax_crawler.py）
- 分析Network请求
- 找到真实API接口

- 参数加密破解

### 3. 反爬虫对抗（03_anti_spider.py）
- User-Agent伪装
- Cookie/Session管理
- IP代理池
- 请求频率控制
- 验证码处理

## 🔥 常见动态网页场景

- 电商网站（商品详情动态加载）
- 社交媒体（无限滚动）
- 新闻网站（评论异步加载）
- 搜索引擎（搜索结果动态生成）

## 💡 技术选型建议

```
简单动态页面      → 分析Ajax接口（最快）
                  ↓
复杂交互          → Selenium（模拟浏览器）
                  ↓
高性能需求        → Playwright（速度快）
                  ↓
严格反爬虫        → 组合使用多种技术
```

## ⚠️ 注意事项

1. **性能问题**：Selenium较慢，优先考虑Ajax
2. **资源消耗**：浏览器占用内存大
3. **稳定性**：需要处理各种异常
4. **合法性**：遵守robots.txt和网站条款
