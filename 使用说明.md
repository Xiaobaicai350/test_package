# 腾讯云转码工具 - 使用说明

## 首次运行说明（macOS 用户必读）

由于 macOS 的安全机制（Gatekeeper），从网络下载的应用可能会被阻止运行。请按照以下步骤操作：

### 方法一：使用快速修复脚本（最简单，推荐）

如果收到了 `快速修复.command` 文件：
1. 双击 `快速修复.command` 文件
2. 脚本会自动移除隔离属性并打开应用

### 方法二：右键打开

1. 找到 `腾讯云转码工具.app` 文件
2. **不要直接双击**，而是**右键点击**应用
3. 选择 **"打开"**（Open）
4. 如果弹出安全提示，点击 **"打开"** 确认

### 方法三：系统设置中允许

如果方法一不行，可以：

1. 打开 **系统设置**（System Settings）
2. 进入 **隐私与安全性**（Privacy & Security）
3. 找到被阻止的应用提示
4. 点击 **"仍要打开"**（Open Anyway）

### 方法四：使用终端命令移除隔离属性

在终端中执行以下命令：

```bash
xattr -cr "/path/to/腾讯云转码工具.app"
```

然后将应用拖到终端中替换 `/path/to/` 部分，或者手动输入应用的完整路径。

## 工具功能

本工具用于处理腾讯云相关的 Excel 数据文件：

- **功能**：将 Excel 文件第四列（数据列）的 Base64 编码内容进行解码，并在第五列显示解码后的内容
- **支持格式**：.xlsx 和 .xls
- **输出文件**：自动生成，格式为 `原文件名_转码_时间戳.xlsx`

## 使用步骤

1. 点击 **"选择 Excel 文件"** 按钮，选择要处理的文件
2. 确认文件路径显示正确
3. 点击 **"开始转码"** 按钮执行处理
4. 处理完成后，查看处理结果和输出文件路径

## 示例文件

工具内置了转换前后的示例文件，可以点击：
- **"查看转换前示例"** - 查看原始 Excel 文件格式
- **"查看转换后示例"** - 查看转换后的结果格式

## 注意事项

- Excel 文件应包含表头（时间、通讯类型、Topic、数据），数据从第二行开始
- 如果遇到问题，请检查 Excel 文件格式是否正确
- 输出文件会保存在输入文件的同一目录下

## 常见问题

**Q: 为什么应用无法打开？**  
A: 这是 macOS 的安全机制。请按照上面的"首次运行说明"操作。

**Q: 处理失败怎么办？**  
A: 请检查 Excel 文件格式是否正确，确保包含4列数据（时间、通讯类型、Topic、数据）。

**Q: 支持哪些 Excel 格式？**  
A: 支持 .xlsx（Excel 2007+）和 .xls（Excel 2003）格式。
