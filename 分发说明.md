# 用户礼包发放工具 - 分发说明

## 分发文件

请将 `dist/用户礼包发放工具.app` **整个文件夹**发送给其他Mac用户。

⚠️ **注意**：必须发送整个 `.app` 文件夹，不能只发送里面的文件。

## 使用方法

### 方法一：直接运行（推荐）

1. 收到 `用户礼包发放工具.app` 文件后，将其放到任意位置（如桌面或应用程序文件夹）
2. **右键点击** `用户礼包发放工具.app`
3. 选择 **"打开"**
4. 如果弹出安全提示，点击 **"打开"** 按钮

### 方法二：如果遇到安全提示

如果双击后提示"无法打开，因为来自身份不明的开发者"：

1. 打开 **系统设置** → **隐私与安全性**
2. 在"安全性"部分，找到被阻止的应用
3. 点击 **"仍要打开"** 按钮

### 方法三：移除隔离属性（高级）

如果以上方法都不行，可以在终端执行：

```bash
xattr -cr /path/to/用户礼包发放工具.app
```

然后双击运行即可。

## 打包命令

如果需要重新打包，执行：

```bash
cd /Users/edy/PycharmProjects/PythonProject
python build_user_pack_sender.py
```

打包后的文件在 `dist/用户礼包发放工具.app`

## 注意事项

1. ✅ 应用已包含所有依赖，无需安装Python环境
2. ✅ 可以在任何Mac系统上运行（macOS 10.13+）
3. ⚠️ 首次运行可能需要用户手动允许（因为未经过Apple代码签名）
4. ⚠️ 如果应用需要网络访问，确保Mac的防火墙设置允许
